# Image to PDF Converter Pro

Image to PDF Converter Pro is a desktop application built with Python that allows users to convert multiple images into a single PDF document.  
The application provides an intuitive graphical interface with features such as drag-and-drop image selection, preview thumbnails, image reordering, optional compression, and PDF-to-image conversion.

This project demonstrates practical desktop application development using Python.

---

## Features

- Convert multiple images into a single PDF
- Drag and drop images into the application
- Preview images before conversion
- Reorder images to control PDF page order
- Optional image compression
- Convert PDF files to images
- Dark mode support
- Progress bar during conversion
- Clean graphical user interface

---

## Technologies Used

- Python
- Tkinter
- Pillow
- TkinterDnD2
- pdf2image
